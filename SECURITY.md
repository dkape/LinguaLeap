# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please do **NOT** open a public issue.

Instead, please report it via email to:
**security@lingualeap.local** (Replace with actual email if available)

We will respond as soon as possible to verify and address the issue.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Credentials & Secrets
This repository is open source. **Never commit real secrets, API keys, or production passwords to this repository.**
- Use `.env` files (gitignored).
- Use GitHub Secrets for CI/CD actions.
